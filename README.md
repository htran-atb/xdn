# xdn

A cheapo CDN.

- Files in github
- Delivered by jsDelivr
- Example: https://cdn.jsdelivr.net/gh/htran-atb/xdn@main/hello.js
- jsDelivr caches contents. [To purge cache](https://www.jsdelivr.com/tools/purge)
